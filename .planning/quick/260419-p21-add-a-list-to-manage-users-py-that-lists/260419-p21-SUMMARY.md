## What Was Built

Added a `list` subcommand to the operator CLI (`scripts/manage_users.py`) that:

- Calls `UserRegistry.load()` to retrieve all registered users
- Prints `"No users registered."` when the registry is empty (exit 0)
- Prints a formatted table with truncated uid (first 8 chars + `...`), name, and status columns for non-empty registries
- Wired into `main()` as `elif subcommand == "list": sys.exit(cmd_list())`
- Appears in `usage()` output and the module docstring

Five new tests cover: empty registry, pending user display, active user status, MAX_USERS rows, and uid truncation.
